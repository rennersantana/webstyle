What is Client-side Programming?


Client-side programming stand for!

client side program is the front-end that runs on a client it is interactive and can run with different functions and dinamicly execute information on the screen

Client-side programming can also send information to a server such as retrieving data from a sever, save datas in the data bases and execute them when needed.


Client side is the structure for your website content.

And it uses the following structures:

  • css Is used to create a layout and design for the html elements on your page. You can also use css for animations these days.
  • JavaScript can do much more. With javaScript can:
  • apply changes to the html (technically you can do everything that css does with javaSript. Don’t do that tough! css is much better for that). You can also apply changes to other elements displayed on a webpage (like interactive svg’s). apply some logic to your webpage by processing data (example: click a button to open a navigation menu. This can get much more complex than that tough. You could write code to programatically resolve a sudoku for example.) receiving and/or submitting data from the browser to/from a server (then usually process it and apply changes to the html).
